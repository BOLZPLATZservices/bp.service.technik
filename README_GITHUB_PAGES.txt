Warum war der Kalender schlecht sichtbar?
Google Calendar Appointment Scheduling lässt sich im eingebetteten iframe nicht frei dunkel gestalten.
Die schwarze Optik wurde vorher per CSS-Filter erzwungen. Das verschlechtert Kontrast und Lesbarkeit.
Die neue Version nutzt deshalb einen schwarzen Rahmen, aber lässt den Kalender selbst hell und klar lesbar.
